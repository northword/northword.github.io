关于软件版本的选择建议

CC 2018 支持各类新硬件、新特性，对硬件有较高的要求。
极不推荐老旧设备上跑新版本，主观上的使用感受不如旧版本。

高版本最低配置
内存至少8GB、建议16GB及以上。
显存至少2GB、建议3GB及以上。
硬盘至少SSD固态硬盘、建议NVME固态硬盘# 推荐三星PM951、PM961、PM981、950PRO、960PRO及以上。
软件缓存所在分区（默认为系统盘C:）保留足够的剩余空间（保持100G以上，尤其AE、PR）根据实际需要更多和高速读写支持。
内存、缓存空间不足，可制约您的高档CPU、出现低负载多核围观的场面。

请务必了解 ，高版本Adobe在处理复杂文件时需较高的配置以满足软件的内存和高速缓存请求、保证软件的响应速度。
建议1080P分辨率以上显示设备、推荐配备数位板/触摸屏。新版本对4K以上分辨率的新机型有更好的支持。
在出现迷之BUG时，请首先考虑在软件中关闭GPU硬件加速(包括 Camera Raw 的硬件加速)。

类似after effort之类的有可能安装大量拓展或脚本的，尽可能使用英文，无论操作界面还是输入内容，尽可能英文，出现中文尽可使用图片，因为部分字体编码有问题会被ae报错造成无法保存无法退出。

尽可能保持各软件大版本一致，以方便使用Adobe动态链接。

安装

使用各路大神制作的
下载对应版本的文件解压缩运行即可，必要时运行破解器。

使用嬴政天下制作的
分为独立版压缩包和大师版镜像分卷包，独立版只有一个软件，大师版集成所有软件

  • 独立版直接下载安装包解压缩运行即可。
  • 大师版镜像分卷包需下载全部压缩包，然后使用文件合并工具合并各个分卷，得到一个RAR文件，
    解压缩，得到一个iso文件，使用虚拟光驱工具加载iso，运行autorun。
    具体参考该目录内的README.TXT 。

关于系统类型

部分软件已经不支持32位操作系统，如果你使用32位操作系统，自行搜索下载低版本的软件。
如pr，ae等，可用CS4及以下版本
ps，ai，aw等仍可使用较新版本。

系统类型可在计算机——右击——属性或设置——系统——关于里查看。
64位下载X64,32位下载X86。

故障排除

1丢失运行库：

如果软件安装运行过程中弹窗找不到MSVCP140.dll，代码已终止运行。以及类似的提示，是缺少运行库。

msvcp、msvcr、vcomp140.dll属于VC++2015版
msvcp、msvcr、vcomp120.dll属于VC++2013版
msvcp、msvcr、vcomp110.dll属于VC++2012版
msvcp、msvcr、vcomp100.dll属于VC++2010版
msvcp、msvcr、vcomp90.dll属于VC++2008版
msvcp、msvcr60、71和80.dll，以及vcomp.dll# 不带数字版本号属于VC++2005版

除了丢失、缺少这类错误之外，如果一个程序打开之后提示“并行配置错误”，那么也是因为没有安装VC++运行库。
如果不会或找不到对应版本的运行库，可以到文件夹06下载运行库合集安装运行。

文件夹06的运行库合集包括：

Microsoft Visual C++ 2008 SP1(9.0.30729.7523)
Microsoft Visual C++ 2010 SP1(10.0.40219.473)
Microsoft Visual C++ 2012 up4(11.0.61135)
Microsoft Visual C++ 2013(12.0.40664)
